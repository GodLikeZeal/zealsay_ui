# zealsay_ui
ui for zealsay

## 运行条件

需要先安装`node`环境和`npm`,尽量不要使用`cnpm`,但是可以使用淘宝得国内镜像。

`$ npm config set registry=http://registry.npm.taobao.org` 配置镜像站


## 运行项目

`$ npm i`  安装依赖

`$ npm run dev`  运行项目


## 开发规范

需要先安装`vue.js`环境，使用`vue.js 2.x`以上。

`$ npm install vue -g` 

`-g` 表示`globel`安装`vue`为全局环境。
